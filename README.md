## 🏭 Industrial Robotics & Automation (ABB RobotStudio)

* 📦 **[Parametric Palletizing Cell](./ABB-Paletizado-Parametrico)**
  * **Description:** A highly scalable, parametric RAPID program for an ABB IRB140 robot that calculates dynamic drop-off positions for interlocking pallet layers using Modulo logic and Euler angles.
  * **Skills:** RAPID, Kinematics, Dynamic TCP, Parametric Logic.

* 🤖 **[Industrial Sorting Cell (Sensor-in-hand)](./ABB-Celda-Clasificacion)**
  * **Description:** A robotic sorting simulation using a vacuum gripper and a laser distance sensor to classify objects based on millimeter-level height differences (18mm vs 19.5mm).
  * **Skills:** Sensor Integration, Threshold Logic, Troubleshooting, Safety Analysis.

## ⚙️ Embedded Systems & Hardware Development

* 🚀 **[Escalator Automated Control System (STM32 Nucleo)](https://github.com/carlos-26/tdse-tp_final_grupo04_isaac_carlos.git)**
  * **Description:** Developed an automated, energy-efficient control system for an escalator using an STM32 Nucleo microcontroller. The project integrates presence sensors to trigger the motor's start-up sequence and manages a safe auto-shutdown delay when no passengers are detected, optimizing energy consumption and mechanical wear.
  * **Skills:** C/C++, STM32 Microcontrollers, Sensor Integration, State Machines, Industrial Control Logic.
